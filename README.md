## GitHub Pages 设置

如果 Pages 还没有开启：

1. 进入仓库 `Settings`
2. 左侧选择 `Pages`
3. `Source` 选择 `Deploy from a branch`
4. `Branch` 选择 `main` 和 `/root`
5. 保存后等待 1 到 2 分钟

部署完成后，根路径会自动打开 `index.html`。
